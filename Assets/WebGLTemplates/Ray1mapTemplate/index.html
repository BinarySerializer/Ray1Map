Controls
Camera

Right mouse button: Pan
WASD: Pan
Mousewheel: Zoom

Objects

Left mouse button: Select
Hold left mouse button: Drag
+/-: Change animation frame
U/J: Move object up/down (isometric)
Free camera mode
(isometric games)

SHIFT: toggle camera movement on/off
or hold the right mouse button

WASD: Move around
Q/Z or NumPad8/2: Move up/down
+/-: Change camera speed
SHIFT in hold mode: Speed boost

Toggle shortcuts (on/off)

C: Show tile collision
B: Show object collision
L: Show object links
O: Show objects
T: Show tiles
P: Animate objects
Y: Animate tiles
G: Show Always objects
E: Show Editor objects
R: Show Rayman object
X: Show object offsets
F: Free camera mode
